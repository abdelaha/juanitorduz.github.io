Fibonacci NumbersÂ¶

We write a recursive function to get the Fibonacci Numbers. Recall that they are defined by \(Fibonacci(1) = Fibonacci(2) = 1\) and

$$ Fibonacci(x) = Fibonacci(x - 1) + Fibonacci(x - 2), $$

for \(x > 2\).

In [3]:
Fibonacci <- function(x) {
    
    ifelse(test = (x < 3),
           yes = 1, 
           no = (Fibonacci(x - 1) + Fibonacci(x - 2)))
}

\(\pi\) EstimationÂ¶

We estimate the value of \(\pi\) using the relation (Basel Problem)

$$ \lim_{N\rightarrow\infty}\sum_{i=1}^N \frac{1}{n^2} = \frac{\pi^2}{6}. $$

Let us write an estimation funtion (using the great pipe operator %>%),

In [6]:
PiEstimateFromBaselProblem <- function(N){
    
    inverse.sq.sum <- 1:N %>% sapply(FUN = function(i) 1/(i)^2) %>% sum
                                     
    pi.estimate <- (inverse.sq.sum * 6) %>% sqrt
                                     
    return(pi.estimate)
}

We now want to plot the estimations ussing ggplot2.
